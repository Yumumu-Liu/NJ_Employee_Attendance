\subsection{系统简介}\label{ux7cfbux7edfux7b80ux4ecb}

这是一个现代化的员工考勤系统，通过人脸识别 + PIN
码来记录员工的上下班打卡。

\textbf{URL}: 打卡页面地址 （由管理员提供）

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\subsection{快速开始}\label{ux5febux901fux5f00ux59cb}

\subsubsection{步骤
1：进入打卡页面}\label{ux6b65ux9aa4-1ux8fdbux5165ux6253ux5361ux9875ux9762}

打开浏览器，访问打卡系统链接。

\subsubsection{步骤
2：选择语言}\label{ux6b65ux9aa4-2ux9009ux62e9ux8bedux8a00}

右上角可选择 \textbf{EN}（英文）或 \textbf{中文}

\subsubsection{步骤
3：选择你的头像}\label{ux6b65ux9aa4-3ux9009ux62e9ux4f60ux7684ux5934ux50cf}

\begin{itemize}
\tightlist
\item
  页面显示两个区域：\textbf{Full-Time}（全职）和
  \textbf{Part-Time}（兼职）
\item
  找到你的名字下的鳄鱼头像
\item
  点击头像卡片
\end{itemize}

\subsubsection{步骤 4：输入 PIN
码}\label{ux6b65ux9aa4-4ux8f93ux5165-pin-ux7801}

\begin{itemize}
\tightlist
\item
  在右侧弹出的窗口中输入你的 4 位 PIN 码
\item
  数字键盘会显示 1-9、0 和清空按钮
\item
  PIN 显示为 •••• （隐藏）
\end{itemize}

\subsubsection{步骤
5：拍照确认}\label{ux6b65ux9aa4-5ux62cdux7167ux786eux8ba4}

\begin{itemize}
\tightlist
\item
  摄像头会启动，显示你的实时镜像画面
\item
  确保你的脸清晰可见
\item
  摄像头是镜像显示，就像照镜子一样
\end{itemize}

\subsubsection{步骤
6：选择打卡类型并提交}\label{ux6b65ux9aa4-6ux9009ux62e9ux6253ux5361ux7c7bux578bux5e76ux63d0ux4ea4}

\begin{itemize}
\tightlist
\item
  \textbf{上班打卡} (Check-In): 按这个按钮记录上班时间
\item
  \textbf{下班打卡} (Check-Out): 按这个按钮记录下班时间
\item
  成功后会显示绿色提示信息
\end{itemize}

\subsection{常见问题}\label{ux5e38ux89c1ux95eeux9898}

\subsubsection{Q: 打卡失败显示''Invalid
PIN''}\label{q-ux6253ux5361ux5931ux8d25ux663eux793ainvalid-pin}

\textbf{A:} PIN 码输入错误。请确保输入的是正确的 4
位数字。如遗忘，请联系管理员。

\subsubsection{Q:
摄像头显示''无法访问摄像头''}\label{q-ux6444ux50cfux5934ux663eux793aux65e0ux6cd5ux8bbfux95eeux6444ux50cfux5934}

\textbf{A:} 1. 检查浏览器是否有摄像头权限 2. 允许网站访问你的摄像头 3.
重新刷新页面

\subsubsection{Q:
拍照后没有反应}\label{q-ux62cdux7167ux540eux6ca1ux6709ux53cdux5e94}

\textbf{A:} 这是正常的。系统正在处理，请等待 2-3 秒。

\subsubsection{Q:
需要切换到后台管理员模式}\label{q-ux9700ux8981ux5207ux6362ux5230ux540eux53f0ux7ba1ux7406ux5458ux6a21ux5f0f}

\textbf{A:} 点击左上角的 \textbf{Admin}
链接进入管理后台（需要输入管理员密码）。

\begin{center}\rule{0.5\linewidth}{0.5pt}\end{center}

\subsection{隐私说明}\label{ux9690ux79c1ux8bf4ux660e}

\begin{itemize}
\tightlist

\item

\item
  请妥善保管你的 PIN 码，不要与他人共享
\end{itemize}
