## 📊 考勤记录管理

### 查看考勤记录
1. 点击 **Attendance Records** 标签
2. 选择日期（可用日期导航器）
3. 表格显示当天所有员工的打卡情况

### 表格说明

| 列 | 说明 |
|------|------|
| **Employee Name** | 员工姓名 |
| **Check-In Time** | 上班打卡时间（HH:mm:ss） |
| **Check-Out Time** | 下班打卡时间（HH:mm:ss） |
| **Status** | 考勤状态：Normal / Missing Check-In / Missing Check-Out / Absent |
| **Actions** | ✕ 按钮删除该条记录 |

### 编辑打卡记录

**修改打卡时间：**
1. 点击表格中的时间（例如 "13:07:42"）
2. 编辑模态框出现
3. 修改：
   - **Type**: Check In 或 Check Out
   - **Date & Time**: 使用日期时间选择器
4. 点 **Update** 保存

**删除打卡记录：**
1. 点击表格右侧的 **✕** 按钮
2. 确认删除
3. 记录被删除（无法恢复）

### 导出数据

**导出为 Excel：**
1. 选择要导出的日期
2. 点击右上角 **Export to Excel** 按钮
3. 文件自动下载：`attendance_YYYY-MM-DD.xlsx`

Excel 文件包含：
- 员工姓名
- 考勤日期
- 上班时间
- 下班时间
- 考勤状态

## 👥 员工管理

### 查看员工列表
1. 点击 **Employees** 标签
2. 显示所有员工及其信息：
   - 姓名
   - PIN 码
   - 工作类型（Full-Time / Part-Time）
   - 打卡开始时间（可选）

### 添加新员工

**方式 1：手动添加**
1. 点击 **Add Employee** 按钮
2. 表单显示以下字段：
   - **Employee Name**: 员工姓名（必填）
   - **PIN**: 4 位数字（必填）
   - **Work Type**: 选择 Full-Time 或 Part-Time
   - **Work Start Time**: 预期打卡时间（可选）

3. 生成 PIN：
   - 点 **Generate** 按钮自动生成随机 4 位 PIN
   - 或手动输入 PIN 码

4. 点 **Save** 创建员工

### 编辑员工信息

**修改现有员工：**
1. 在员工列表中，点击员工名字或 **✏️** 按钮
2. 编辑表单出现，显示：
   - 当前信息
   - **Current PIN: XXXX** 提示
3. 修改任何字段
4. 点 **Generate** 或 **Change** 生成新 PIN（可选）
5. 点 **Update** 保存更改

### 删除员工

**软删除：**
1. 在员工列表中找到员工
2. 点 **✕** 删除按钮
3. 确认删除
4. 员工被标记为不活跃（数据保留，不会在打卡页面显示）

### 批量导入员工

**导入 Excel 文件：**
1. 点 **Import Employees** 按钮
2. 选择 Excel 文件（.xlsx, .xls, .csv）
3. 文件格式要求：

| 列名 | 说明 | 必填 |
|------|------|------|
| **Employee Name** 或 **员工姓名** | 员工名字 | ✓ |
| **PIN** 或 **pin** | 4 位数字 PIN | ✓ |
| **Work Type** 或 **工作类型** | full 或 part | ✗ |
| **Work Start Time** 或 **打卡开始时间** | HH:mm 格式 | ✗ |

**导入示例 Excel：**
```
Employee Name | PIN  | Work Type | Work Start Time
张三          | 1234 | full      | 09:00
李四          | 5678 | part      | 14:00
王五          | 0000 | full      |
```

4. 点 **Upload** 导入
5. 系统显示导入结果：成功数、错误数

### 下载导入模板

1. 在员工管理页面，点 **Download Template**
2. 获得标准 Excel 模板
3. 填入员工信息后上传

## 🆘 常见问题

### Q: 忘记管理员密码
**A:** 修改 `.env` 文件中的 `ADMIN_PASSWORD` 值，重新部署应用。

### Q: 导入员工失败
**A:** 检查：
- Excel 文件格式正确
- PIN 码必须是 4 位数字
- Work Type 必须是 "full" 或 "part"

### Q: 考勤记录无法编辑
**A:** 检查：
- 已登录管理后台
- 记录确实存在于表格中
- 点击正确的时间单元格

### Q: 员工删除后能否恢复
**A:** 软删除后数据仍在数据库中，但不会显示。无法通过 UI 恢复，需要数据库管理员手动处理。

### Q: 如何导出所有数据
**A:** 
- 按日期导出 Excel（当前方式）
- 或联系 Supabase 获取数据库备份
